# PRD: 농산물 직거래 플랫폼

## 1. 개요

### 1.1 프로젝트명
**밭뜰 (BatTteul)** - 밭에서 뜰로, 신선한 농산물 직거래 플랫폼

### 1.2 목적
농부가 직접 재배한 신선한 과일, 채소 등 농산물을 소비자에게 직접 연결해주는 플랫폼 서비스

### 1.3 핵심 가치
- 중간 유통 없이 농부와 소비자 직접 연결
- 신선한 농산물 정보 제공
- 간편한 문의 연결 (전화/문자)

---

## 2. 대상 사용자

### 2.1 판매자 (농부)
- 소규모 농가 운영자
- 직접 재배한 농산물을 판매하고 싶은 개인
- 온라인 판매 채널이 필요한 농업인

### 2.2 구매자 (소비자)
- 신선한 농산물을 직거래로 구매하고 싶은 일반 소비자
- 생산자 정보를 확인하고 싶은 소비자
- 지역 농산물에 관심 있는 소비자

---

## 3. 핵심 기능

### 3.1 판매자 기능

#### 3.1.1 상품 등록
- 상품명
- 상품 설명
- 가격 (협의 가능 표시 옵션)
- 상품 이미지 (최대 5장)
- 카테고리 선택 (과일/채소/곡물/기타)
- 재배 지역
- 수확 시기
- 판매 수량/단위 (예: 5kg, 한 박스)

#### 3.1.2 판매자 프로필
- 농장/판매자 이름
- 농장 소개
- 농장 위치 (시/군/구 단위)
- **문의 휴대폰 번호** (필수)
- 농장 사진 (선택)

#### 3.1.3 상품 관리
- 등록 상품 목록 조회
- 상품 수정/삭제
- 판매 상태 변경 (판매중/품절/판매종료)

### 3.2 구매자 기능

#### 3.2.1 상품 탐색
- 전체 상품 목록 조회
- 카테고리별 필터링
- 지역별 필터링
- 최신순/가격순 정렬
- 검색 기능 (상품명, 판매자명)

#### 3.2.2 상품 상세 조회
- 상품 정보 전체 확인
- 판매자 정보 확인
- **판매자 연락처로 전화/문자 연결**

#### 3.2.3 관심 상품
- 관심 상품 저장
- 관심 상품 목록 조회

### 3.3 공통 기능

#### 3.3.1 회원 관리
- 회원가입 (이메일/소셜 로그인)
- 로그인/로그아웃
- 프로필 수정
- 회원 탈퇴

#### 3.3.2 알림 (선택적 구현)
- 관심 판매자 신규 상품 등록 알림
- 문의 받음 알림 (판매자)

---

## 4. 페이지 구성

### 4.1 메인 페이지
- 최신 등록 상품
- 인기 상품 (조회수 기준)
- 카테고리 바로가기
- 검색 바

### 4.2 상품 목록 페이지
- 상품 카드 그리드 뷰
- 필터/정렬 옵션
- 무한 스크롤 또는 페이지네이션

### 4.3 상품 상세 페이지
- 상품 이미지 갤러리
- 상품 정보
- 판매자 정보 카드
- **연락하기 버튼** (전화 걸기 / 문자 보내기)

### 4.4 판매자 페이지
- 판매자 프로필
- 판매자의 전체 상품 목록
- 연락처 정보

### 4.5 마이페이지
- 내 정보 관리
- (판매자) 내 상품 관리
- (구매자) 관심 상품 목록

### 4.6 상품 등록/수정 페이지
- 상품 정보 입력 폼
- 이미지 업로드

## 5. 데이터 모델

### 5.1 User (사용자)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | 고유 식별자 |
| email | string | 이메일 |
| name | string | 이름 |
| phone | string | 휴대폰 번호 |
| userType | enum | buyer/seller |
| profileImage | string | 프로필 이미지 URL |
| createdAt | datetime | 가입일 |

### 5.2 Seller (판매자 정보)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | 고유 식별자 |
| userId | string | User FK |
| farmName | string | 농장명 |
| farmDescription | string | 농장 소개 |
| region | string | 지역 |
| contactPhone | string | 문의 전화번호 (필수) |
| farmImages | string[] | 농장 이미지 URLs |

### 5.3 Product (상품)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | 고유 식별자 |
| sellerId | string | Seller FK |
| title | string | 상품명 |
| description | string | 상품 설명 |
| price | number | 가격 |
| priceNegotiable | boolean | 가격 협의 가능 여부 |
| category | enum | fruit/vegetable/grain/other |
| region | string | 재배 지역 |
| harvestDate | date | 수확 시기 |
| quantity | string | 수량/단위 |
| images | string[] | 상품 이미지 URLs |
| status | enum | available/soldout/closed |
| viewCount | number | 조회수 |
| createdAt | datetime | 등록일 |
| updatedAt | datetime | 수정일 |

### 5.4 Favorite (관심 상품)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | 고유 식별자 |
| userId | string | User FK |
| productId | string | Product FK |
| createdAt | datetime | 등록일 |

- 모바일 환경 최적화 필수 (전화/문자 연결 기능)
